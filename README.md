Affordmed Full-Stack URL Shortener
Overview

This project is a Full-Stack URL Shortener designed for the Affordmed Campus Hiring Evaluation. It consists of:

Logging Middleware: Reusable middleware for backend and frontend logging.

Backend: Node.js + Express microservice for URL shortening, redirection, and analytics.

Frontend: React + Material UI application to shorten URLs and view statistics.

The application uses in-memory storage for demonstration and evaluation purposes.

Folder Structure
/your-repo-rollno
  /logging-middleware
  /backend
  /frontend

logging-middleware: Reusable logging middleware. Logs are sent to Affordmed logging API.

backend: Express backend microservice.

frontend: React frontend using Material UI.
